HTML e CSS

HTML e CSS são usados para fazer sites

HTML é uma linguagem de marcação usada para estruturar o conteúdo da web, enquanto CSS é uma linguagem de estilo usada para definir a aparência visual desse conteúdo.

HTML (Hypertext Markup Language) é a linguagem padrão para criar páginas

CSS (Cascading Style Sheets) é uma linguagem de estilo usada para descrever a apresentação de um documento HTML.

Tags

As tags HTML são usadas para estruturar e formatar o conteúdo de uma página web. Elas são elementos que definem diferentes partes do documento e geralmente vem em pares de abertura e fechamento.

Exemplos de tags HTML comuns incluem:

  • A tag '< b >' é utilizada para deixar o texto em negrito
  • A tag '< i >' é utilizada para deixar o texto em itálico
  • A tag '< u >' é utilizada para deixar o texto sublinhado
  • A tag '< strong >' é utilizada para deixar o texto com forte ênfase
  • A tag '< em >' é utilizada para deixar o texto com ênfase
  • A tag '< span >' é utilizada para agrupar elementos em linha, permitindo aplicar estilos específicos a uma parte do texto. Exemplo: Texto em span
  • A tag '< div >' é utilizada para agrupar elementos em bloco, permitindo aplicar estilos ou scripts a uma seção inteira da página. Exemplo:
